# README

## Setup

```
$ cd webApp
$ npm install
```

## Compiling Javascript

```
$ cd webApp
$ npm run compile-test
```

## Running Tests

To manually run them in the browser:

```
$ cd webApp
$ open -a /Applications/Google\ Chrome.app/ test.html
```

To run with karma: 

```
$ cd webApp
$ node_modules/.bin/karma start karma.conf.js
```
